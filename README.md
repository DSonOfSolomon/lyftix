# Lifytix

Lifytix is a self-hosted personal analytics platform.

It tracks coding activity, workouts, daily check-ins, GitHub activity, and system metrics.

## Tech Stack

- Spring Boot
- PostgreSQL
- Python workers
- Docker
- Prometheus
- Grafana
- Tailscale

## Project Structure

- backend: Spring Boot API
- workers: Python ingestion jobs
- infrastructure: Docker, monitoring, and server setup files
